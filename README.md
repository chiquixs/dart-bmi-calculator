This project is a simple Body Mass Index (BMI) Calculator built using the Dart programming language.  
The application allows users to input their height and weight, calculate BMI, determine BMI categories, and store calculation history

This project was also developed using a Git branching workflow, where each feature was implemented in separate branches before being merged into the main branch.

---

## Features

- Input height and weight
- BMI calculation
- BMI category classification
- Input validation
- Calculation history
- Loop for multiple calculations


The project is developed using feature branches:

- 1. input-validation  
  Handles user input and validation for height and weight.

- 2. bmi-calculation  
  Performs BMI calculation and determines the BMI category.

- 3. history-feature  
  Stores and displays BMI calculation history using loops.

All branches are merged into the main branch after the features are completed
